메가박스 웹사이트 클론코딩 입니다.

 ## 📚Technology Stack
* HTML
* CSS
* JS
* jQuery

## ⚙Main Function
1. HTML,CSS,JS,jQuery를 활용하여 만든 반응형 메가박스 클론코딩 프로젝트입니다.
2. swiper js를 활용한 이미지 슬라이드 구현
3. jQuery를 활용한 탭 구현
4. YouTube API를 사용해 동영상 모달창 구현
